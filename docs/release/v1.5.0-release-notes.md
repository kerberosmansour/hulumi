# v1.5.0 Release Notes

This is the maintainer-facing release note draft for the Hulumi v1.5.0 npm publish.
The public changelog source of truth is [`CHANGELOG.md`](../../CHANGELOG.md).

## Package set

Publish all six packages atomically at `1.5.0`:

- `@hulumi/baseline`
- `@hulumi/policies`
- `@hulumi/drift`
- `@hulumi/k8s-baseline`
- `@hulumi/cloudflare-baseline`
- `@hulumi/platform-patterns`

Every package continues to use npm trusted publishing and SLSA Build L3 provenance.

## Release theme

v1.5.0 turns Hulumi's hardened component set into a cloud-platform operating model:

- AWS organization guardrails and the matching `HulumiAwsOrgHardeningPack`.
- Hardened S3/KMS Pulumi state backend posture and `STATE-1`.
- EKS cluster create/adopt foundation and foundation-specific EKS policy checks.
- Read-only `hulumi validate live` JSON, Markdown, and SARIF posture artifacts.
- AWS secure primitives for deployment roles, workload roles, Secrets Manager secrets,
  and launch templates.
- Routed `SecurityDetectionFoundation` alarm families and `DETECT-*` checks.
- GitHub runner-governance component, linter checks, live-validator evaluator, and
  deployment-governance policy backstops.

## Security advisories

No new consumer-facing GitHub Security Advisories are planned for v1.5.0. The release
is proactive hardening and new policy coverage rather than a disclosure of a newly
patched downstream package vulnerability.

The existing `pnpm audit --prod` findings are transitive Pulumi/tooling advisories in
`tmp`, `brace-expansion`, and `protobufjs`. They remain tracked as dependency hygiene,
not as v1.5.0 release blockers.

## Pre-tag gates

Run the normal release checklist before tagging:

```bash
pnpm install
pnpm -r build
pnpm -r typecheck
pnpm -r test
pnpm -r lint
pnpm run lint:license-boundary
pnpm run lint:exact-pin-guard
pnpm run format:check
```

After the tag workflow publishes, verify all six attestations with:

```bash
VERSION=1.5.0
DEST=$(mktemp -d /tmp/hulumi-release-attestations.XXXXXX)

npm pack "@hulumi/baseline@$VERSION" --pack-destination "$DEST" --json
gh attestation verify "$DEST/hulumi-baseline-$VERSION.tgz" --repo kerberosmansour/hulumi

npm pack "@hulumi/policies@$VERSION" --pack-destination "$DEST" --json
gh attestation verify "$DEST/hulumi-policies-$VERSION.tgz" --repo kerberosmansour/hulumi

npm pack "@hulumi/drift@$VERSION" --pack-destination "$DEST" --json
gh attestation verify "$DEST/hulumi-drift-$VERSION.tgz" --repo kerberosmansour/hulumi

npm pack "@hulumi/k8s-baseline@$VERSION" --pack-destination "$DEST" --json
gh attestation verify "$DEST/hulumi-k8s-baseline-$VERSION.tgz" --repo kerberosmansour/hulumi

npm pack "@hulumi/cloudflare-baseline@$VERSION" --pack-destination "$DEST" --json
gh attestation verify "$DEST/hulumi-cloudflare-baseline-$VERSION.tgz" --repo kerberosmansour/hulumi

```
