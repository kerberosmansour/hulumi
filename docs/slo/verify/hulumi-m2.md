## Coverage gaps

- **Real `pulumi preview --policy-pack`** — requires the Pulumi CLI plus a reachable Pulumi backend, plus a real Pulumi project outside the mock runtime. The M2 contract explicitly scopes this to "mocks-only for every-PR tests; sandbox integration weekly" (critique E3). The mock-runtime smoke test exercises the same construction path the real CLI would, so the gap is narrow. The real-CLI smoke test runs in M3 once the weekly sandbox integration infrastructure lands.
- **S3-backed state with a real live AWS call** — same carve-out; H2's S3 detection was exercised via synthetic `StackValidationArgs`, not a real `pulumi preview` over a live S3 backend. M3 weekly integration will cover the live case.
- **CI execution** — the GitHub remote `kerberosmansour/hulumi` still doesn't exist; CI workflow + exact-pin-guard job + DCO sign-off job are committed but have never run. Tracked alongside the M1 deferral for the first push.
